# football-match-prediction
This project applies multiple machine learning algorithms to predict football match outcomes. It includes data preprocessing, model training, evaluation with metrics (accuracy, log loss, precision, recall, F1-score), confusion matrix visualization, and comparative analysis of all models. 
The dataset used in this project is too large to be stored in the repository.  
Please download it from https://www.kaggle.com/competitions/football-match-probability-prediction/data and place the CSV files in the project folder.
